## 浮点数

浮点数，称为float。
浮点数用 a * b^10 形式的科学计数法标识。比如：3.14，标识为：314e-2或者314E-2。
这些数字在内存中也是按照科学计数法存储。

## 类型转换和四舍五入

1. 类似与int(),我们也可以使用float()讲其他类型转化为浮点数。
2. 整数和浮点数混合运算时，表达式结果自动转型为浮点数。比如：2+8.0的结果是10
3. round（value）可以返回四舍五入的值
    注：但不会改变原有值，而是产生新值。

## 增强型赋值运算符
运算符+、- 、 * 、 / 、 // 、 % 和赋值符结合可以构成增强型复制运算符。
a = a + 1 等价于 a += 1

增强型运算符

| 运算符   | 例子 |     等价| 
| :----- | :--: | -------: |
| += |  a+=2 | a=a+2 |
| －= |  a-=2  | a=a-2 |
| *= |  a*=2  | a=a*2 |
| /= |  a/=2  | a=a/2 |
| //= |  a//=2  | a=a//2 |
| \*\*= |  a**=2  | a=a**2 |
| %= |  a%=2  | a=a%2 |
注：“+=”中间不加空格
