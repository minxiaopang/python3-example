# format()基本用法
Python2.6开始，新增一种格式化字符串函数str.format
(),他增强了字符串格式化的功能。基本语法是通过{}和：来代替以前的%。
format函数可以接受不限个参数，位置可以不按顺序。

示例：
```python
a = '我的名字是:{0},年龄是:{1}'
print(a.format('小胖', '23'))

a = '我的名字是:{0},年龄是:{1},{0}是个好伙子'
print(a.format('小胖', '23'))

c = '名字是:{name},年龄是:{age}'
print(c.format(name='小胖', age=23))

```
